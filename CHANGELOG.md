## [0.0.1] - 2025-02-28

### Added

- Initial production-ready release of the Day and Knight VS Code extension
- Auto update theme based on local time (day: 06:00–17:59, night: 18:00–05:59)
- Random theme selection from configured light/dark theme arrays
- Deterministic daily selection using a hash of the local day string
- Status bar toggle for manual theme switch
- Commands: Toggle Theme, Toggle to Day Theme ☀️, Toggle to Night Theme 🌙, Enable Auto Update, Disable Auto Update
